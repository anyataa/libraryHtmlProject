### This is an application for Library Management System. This Application consist of 2 roles: User and Admin. 

#### User Role:
- Log in
- Look at Book Details, Title, and Image
- Borrow Book

#### Admin Role: 
- Edit Book
- Add Book
- Delete Book

### Notes :
This Application use 1 Outside Library called Light-Slider.
